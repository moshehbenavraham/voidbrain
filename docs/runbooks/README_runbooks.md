# Runbooks

This directory holds operational runbooks for Voidbrain.

## Purpose

- Document repeatable steps for diagnosing and recovering local issues.
- Keep guidance concise, validated, and safe for repository-local workflows.
- Use separate files for separate scenarios.

## Guidance

- Prefer clear checklist-style steps.
- Keep examples synthetic and avoid secrets or private vault content.
- Link to the relevant docs, validation commands, or source files when the
  steps depend on them.
